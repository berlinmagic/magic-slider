Important Settings

Option default Description
slideDirection horizontal The Slide-Direction "horizontal" / "vertical"
carousel false Carousel = Invinite-Scroll .. (Don't slide Back to first Panel when hit the end)
crossLinking false Change URL and linkable Panels .. (Turned off, because I don't like Anchors)
autoCycle false Endless Play
autoSlide false Slider changes automatic = Autoplay
autoSlideStopWhenClicked true Stop Autoplay when Link is clicked
firstPanelToLoad 1 Start-Panel of the given Slider
externalTriggerSelector a.xtrig Selector for external-Link = <a class="xtrig" href="#1" rel="magic-slider-1">text</a>

Slider Effekts

Option default Description
autoHeight true Change the Panel Height for each Panel
autoHeightEaseDuration 1000 Time to animate the Height-Transition .. (in miliseconds)
autoHeightEaseFunction easeInOutExpo Height-Transition-Effekt
autoSlideInterval 7000 Autoplay Stay-Time
slideEaseDuration 1000 Slide-Time .. (in miliseconds)
slideEaseFunction easeInOutExpo Height-Transition-Effekt

Automatic Navigation

Option default Description
dynamicArrows true Generate dynamic (left/right) arrows
dynamicArrowLeftText « Linktext of the left-arrow
dynamicArrows » Linktext of the right-arrow
dynamicTabs true Generate dynamic Tab-Navigation
dynamicTabsAlign center Vertical-Position of the Tab-Navigation
dynamicTabsPosition top Horizontal-Position of the Tab-Navigation
panelTitleSelector h2.title TitleSelector = Content is used to fill Tab-Navigation-Links
